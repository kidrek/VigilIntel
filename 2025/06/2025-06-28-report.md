### <a id="vulnérabilité-critique-dans-citrix-netscaler-adc-et-gateway-cve-2025-5777"></a> Vulnérabilité critique dans Citrix NetScaler ADC et Gateway (CVE-2025-5777)
Une vulnérabilité critique, surnommée "Citrix Bleed 2" (CVE-2025-5777), a été découverte dans les produits Citrix NetScaler ADC et Gateway. Ces produits sont des contrôleurs de livraison d'applications qui optimisent, sécurisent et équilibrent le trafic réseau pour les applications web. La vulnérabilité est une faille de divulgation d'informations (memory overread) pouvant conduire à la divulgation de données sensibles, telles que des jetons de session authentifiés, ce qui pourrait permettre à un attaquant d'obtenir un accès supplémentaire à l'appareil ou aux systèmes. Des signes d'exploitation active de cette vulnérabilité ont été observés, avec une augmentation des sessions suspectes sur les appareils Citrix.
* Publication date : 2025/06/27
* 🔗 Source : https://www.bleepingcomputer.com/news/security/citrix-bleed-2-flaw-now-believed-to-be-exploited-in-attacks/, https://www.cisecurity.org/advisory/multiple-vulnerabilities-in-citrix-products-could-allow-for-disclosure-of-sensitive-data_2025-060
* 📝 CVE : [CVE-2025-5777](https://www.cve.org/CVERecord?id=CVE-2025-5777)
* 🖥️ Impacted Product : Citrix NetScaler ADC, Citrix Gateway
* 💯 CVSS : Critical (d'après l'article, score exact non spécifié mais > 8 implicite)
* 🚨 Security recommandations : Examiner et appliquer les mises à jour et correctifs de sécurité fournis par Citrix dès que possible. Surveiller les activités suspectes sur les appareils NetScaler ADC et Gateway.
* 🚨 Indicator of Compromise :
    *   Sessions suspectes sur les appareils Citrix.

* 💯 CVSS : Critical (d'après l'article, score exact non spécifié mais > 8 implicite)
* 🚨 Security recommandations : Examiner les mises à jour et correctifs de sécurité de Cisco pour ISE et ISE-PIC. Appliquer les correctifs sans délai.

### <a id="vulnérabilité-dexécution-de-code-à-distance-et-de-suppression-de-fichiers-dans-le-plugin-wordpress-game-users-share-buttons-cve-2025-6755"></a> Vulnérabilité d'exécution de code à distance et de suppression de fichiers dans le plugin WordPress Game Users Share Buttons (CVE-2025-6755)
Le plugin WordPress "Game Users Share Buttons" est vulnérable à la suppression arbitraire de fichiers en raison d'une validation insuffisante du chemin de fichier dans la fonction `ajaxDeleteTheme()`. Cette faille, présente dans toutes les versions jusqu'à et y compris la version 1.3.0, permet à des attaquants ayant un accès de niveau abonné (Subscriber) d'ajouter des chemins de fichiers arbitraires (comme `../../../../wp-config.php`) au paramètre `themeNameId` de la requête AJAX. Cela peut entraîner une exécution de code à distance (RCE) via la suppression du fichier de configuration WordPress.
* Publication date : 2025/06/28
* 🔗 Source : https://cvefeed.io/vuln/detail/CVE-2025-6755
* 📝 CVE : [CVE-2025-6755](https://www.cve.org/CVERecord?id=CVE-2025-6755)
* 🖥️ Impacted Product : WordPress Game Users Share Buttons plugin (versions <= 1.3.0)
* 💯 CVSS : 8.8 | HIGH

### <a id="vulnérabilité-de-traversée-de-répertoire-dans-le-plugin-wordpress-beeteam368-extensions-cve-2025-6381"></a> Vulnérabilité de traversée de répertoire dans le plugin WordPress BeeTeam368 Extensions (CVE-2025-6381)
Le plugin WordPress "BeeTeam368 Extensions" est vulnérable à la traversée de répertoire via la fonction `handle_remove_temp_file()`. Cette faille, présente dans toutes les versions jusqu'à et y compris la version 2.3.4, permet à des attaquants authentifiés (avec un accès de niveau abonné ou supérieur) d'effectuer des actions sur des fichiers en dehors du répertoire prévu. Cette vulnérabilité peut être utilisée pour supprimer le fichier `wp-config.php`, ce qui peut entraîner une prise de contrôle du site.
* Publication date : 2025/06/28
* 🔗 Source : https://cvefeed.io/vuln/detail/CVE-2025-6381
* 📝 CVE : [CVE-2025-6381](https://www.cve.org/CVERecord?id=CVE-2025-6381)
* 🖥️ Impacted Product : WordPress BeeTeam368 Extensions plugin (versions <= 2.3.4)
* 💯 CVSS : 8.8 | HIGH

### <a id="le-ralentissement-de-cloudflare-par-la-russie-rend-des-sites-inaccessibles"></a> Le ralentissement de Cloudflare par la Russie rend des sites inaccessibles

* Publication date : 2025/06/27
* 🔗 Source : https://www.bleepingcomputer.com/news/technology/russias-throttling-of-cloudflare-makes-sites-inaccessible/
* 🎭 Threat Actor : Russie (via les FAI russes)
* 🎯 Threat Target : Sites web et services protégés par Cloudflare, utilisateurs d'Internet en Russie
* ⚔️ Threat Tactic : Throttling (ralentissement) de l'accès internet
* 🚨 Security recommandations : Les organisations opérant en Russie ou ciblant des utilisateurs russes devraient évaluer l'impact potentiel sur l'accessibilité de leurs services. Les utilisateurs russes pourraient envisager des VPN pour contourner les restrictions.
