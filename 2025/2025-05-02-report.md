## Amende GDPR pour TikTok concernant le transfert de données vers la Chine
La Commission irlandaise de protection des données (DPC) a infligé à TikTok une amende de 530 millions d'euros pour non-respect des réglementations GDPR, spécifiquement concernant le transfert des données des utilisateurs de l'Espace Économique Européen (EEE) vers la Chine. 💸
* Publication date : 2025/05/02
* Source : https://thehackernews.com/2025/05/tiktok-slammed-with-530-million-gdpr.html, https://www.bleepingcomputer.com/news/security/tiktok-fined-530-million-for-sending-european-user-data-to-china/
* 🎯 Threat Target : Utilisateurs de TikTok dans l'EEE

## Microsoft rend les nouveaux comptes par défaut sans mot de passe
Microsoft a annoncé que les nouveaux comptes seront par défaut sans mot de passe, encourageant l'utilisation de méthodes d'authentification résistantes au phishing comme les passkeys pour améliorer la sécurité. 🔑
* Publication date : 2025/05/02
* Source : https://thehackernews.com/2025/05/microsoft-sets-passkeys-default-for-new.html, https://www.bleepingcomputer.com/news/microsoft/microsoft-makes-all-new-accounts-passwordless-by-default/
* 🎯 Threat Target : Utilisateurs de comptes Microsoft (nouvellement créés)
* 💡 Security recommandations : Adopter des méthodes d'authentification sans mot de passe lorsque possible.

## Campagne Malware MintsLoader distribuant GhostWeaver via Phishing
Le loader malware MintsLoader est utilisé dans une chaîne d'infection multi-étapes impliquant des scripts obfuscés pour déployer le RAT GhostWeaver. Le malware utilise des techniques d'évasion de sandbox et de machine virtuelle, ainsi que la DGA et TLS pour sa communication C2. 👻🎣
* Publication date : 2025/05/02
* Source : https://thehackernews.com/2025/05/mintsloader-drops-ghostweaver-via.html
* 👹 Threat Actor : Non spécifié (campagne malware)
* 💥 Threat Tactic : Phishing, Évasion, Communication C2 (DGA, TLS)
* 🛠️ Threat Tools : MintsLoader (loader), GhostWeaver (RAT)
* 💡 Security recommandations : Sensibilisation au phishing, surveillance réseau des communications C2 (DGA, TLS anormaux), utilisation d'EDR.

## Plugin de sécurité WordPress frauduleux permettant un accès administrateur à distance
Une nouvelle campagne cible les sites WordPress en distribuant un malware déguisé en plugin de sécurité nommé "WP-antymalwary-bot.php". Ce plugin malveillant permet un accès à distance, se cache du tableau de bord administrateur et exécute du code à distance via une fonctionnalité de ping vers un serveur C2. 🔌️😈
* Publication date : 2025/05/01
* Source : https://thehackernews.com/2025/05/fake-security-plugin-on-wordpress.html
* 👹 Threat Actor : Non spécifié (campagne malware)
* 💥 Threat Tactic : Distribution de malware (via plugin frauduleux), Persistance, Accès à distance, Exécution de code à distance, Communication C2
* 🎯 Threat Target : Sites WordPress
* 🛠️ Threat Tools : WP-antymalwary-bot.php (plugin malveillant)
* 💡 Security recommandations : Vérifier la source des plugins WordPress, supprimer les plugins non fiables, analyser les sites pour détecter les portes dérobées.
* 🔍 Indicator of Compromise :
    * DOMAIN : italicfonts[.]org
