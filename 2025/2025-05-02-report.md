## <a name="wordpress-fake-plugin"></a>👾 Faux Plugin de Sécurité WordPress Permet un Accès Administrateur à Distance
Une nouvelle campagne cible les sites WordPress en déguisant un malware en plugin de sécurité nommé "WP-antymalwary-bot.php" 🛡️. Ce faux plugin inclut des fonctionnalités pour maintenir l'accès, se cacher du tableau de bord administrateur et exécuter du code à distance 😈. Il communique avec un serveur de commande et de contrôle (C&C).
* Publication date : 2025/05/01
* Source : https://thehackernews.com/2025/05/fake-security-plugin-on-wordpress.html
* Victime : Sites WordPress.
* Tactique / Modus operandi : Distribution de malware déguisé en plugin légitime, persistance, exécution de code à distance, communication C2.
* Outils / Malware associé : WP-antymalwary-bot.php.
* Recommandation : N'installer des plugins que depuis des sources fiables et vérifier leur légitimité. Effectuer des analyses de sécurité régulières sur les sites WordPress.
* Observables :
    * DOMAIN : `italicfonts[.]org`
