## Strategic Recommendations

1. **Migrate to FIDO2 Hardware MFA**
   Traditional MFA is bypassed by automated OAuth kits (ConsentFix v3) and Device Code phishing (EvilTokens). Deploy FIDO2/WebAuthn hardware keys (Yubikey) for all administrative and developer accounts to ensure phishing resistance.

2. **Enforce Strict Supply Chain Integrity**
   The 'wormification' of npm (TeamPCP) requires a move beyond simple scanning. Implement a private package registry with mandatory 72-hour delay on new versions and strict '--ignore-scripts' policies for local developer environments.

## Operational Recommendations

1. **Patch cPanel and Limit Management Ports**
   Immediately update cPanel/WHM to version 11.136.0.5 or higher to mitigate CVE-2026-41940. Restrict access to ports 2083/2087 to known administrative IPs via firewall rules.

2. **Audit Entra ID Application Consents**
   Use PowerShell (MgGraph) to identify all non-verified applications with 'Mail.Read' or 'Directory.Read' permissions. Revoke any unknown consents to mitigate ConsentFix v3 and Tycoon2FA risks.
