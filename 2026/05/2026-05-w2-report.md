## Strategic Recommendations

1. **Enforce Phishing-Resistant FIDO2 Hardware MFA**
   Traditional MFA (SMS, TOTP) is systematically bypassed by actors like ShinyHunters and TCLBanker using session hijacking. Deploy FIDO2/WebAuthn hardware keys (YubiKey) for all administrative and developer accounts to prevent credential theft. Revoke all active sessions upon rollout to clear existing infostealer-harvested tokens.

2. **Establish Rigorous Supply Chain Software Auditing**
   Given the TeamPCP 'Mini Shai-Hulud' worm, organizations must implement Sigstore-based signature verification and Software Composition Analysis (SCA) with real-time alerting. Restrict npm/PyPI installations to vetted versions and block auto-updates for critical CI/CD pipelines to prevent viral package contamination.

## Operational Recommendations

1. **Apply Critical Edge Security Patches Immediately**
   Exploitation of CVE-2026-0300 (PAN-OS) and CVE-2026-6973 (Ivanti) is active. Disable Captive Portals and unauthenticated management interfaces on internet-facing devices until patches are applied. Monitor logs for unusual root-level shell activity or unauthorized SOCKS5 tunnels (EarthWorm).

2. **Monitor eBPF Loading for Rootkit Detection**
   The Quasar Linux RAT (QLNX) uses eBPF programs to hide malicious processes and intercept PAM modules. Use 'bpftool' to audit loaded eBPF programs on Linux servers and alert on any unsigned or unexpected hooks into the network stack or authentication process. Rotate SSH keys immediately if PAM tampering is suspected.

## Recommandations Stratégiques

1. **Imposer un MFA matériel FIDO2 résistant au phishing**
   Le MFA traditionnel (SMS, TOTP) est systématiquement contourné par ShinyHunters et TCLBanker via le détournement de session. Déployez des clés matérielles FIDO2/WebAuthn (YubiKey) pour tous les comptes d'administration et de développeurs. Révoquez toutes les sessions actives lors du déploiement pour effacer les jetons volés.

2. **Établir un audit rigoureux des logiciels supply chain**
   Face au ver 'Mini Shai-Hulud' de TeamPCP, les organisations doivent implémenter une vérification de signature basée sur Sigstore. Restreignez les installations npm/PyPI aux versions validées et bloquez les mises à jour automatiques pour les pipelines CI/CD critiques afin d'éviter la contamination virale.

## Recommandations Opérationnelles

1. **Appliquer immédiatement les correctifs de sécurité périmétriques**
   L'exploitation de CVE-2026-0300 (PAN-OS) et CVE-2026-6973 (Ivanti) est active. Désactivez les portails captifs et les interfaces de gestion non authentifiées sur les équipements exposés. Surveillez les logs pour toute activité inhabituelle de shell root ou de tunnels SOCKS5 non autorisés.

2. **Surveiller le chargement eBPF pour détecter les rootkits**
   Quasar Linux RAT (QLNX) utilise des programmes eBPF pour masquer des processus et intercepter les modules PAM. Utilisez 'bpftool' pour auditer les programmes eBPF chargés sur les serveurs Linux. Effectuez une rotation des clés SSH immédiatement si une altération de PAM est suspectée.
