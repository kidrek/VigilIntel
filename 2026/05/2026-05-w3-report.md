**Geopolitical context:** Cyber operations are increasingly synchronized with energy sovereignty, as seen in the Caucasus. ## Strategic Recommendations

1. **Enforce FIDO2 Hardware MFA Organization-Wide**
   Observed Tycoon2FA campaigns use OAuth 2.0 Device Code phishing to bypass legacy TOTP/SMS MFA. Deploy FIDO2/WebAuthn hardware keys (YubiKey 5) to eliminate phishing-susceptible flows and restrict Device Code logins in Entra ID.

2. **Harden CI/CD and npm Supply Chain**
   Given the TanStack and node-ipc incidents affecting OpenAI, implement mandatory Software Composition Analysis (SCA) and use private repositories (Artifactory/Nexus) that mirror and scan external npm packages before deployment.

## Operational Recommendations

1. **Revoke Active OAuth Sessions and Device Registrations**

2. **Patch Quest KACE SMA and Exchange Instantly**
   CVE-2025-32975 and CVE-2026-42897 are under active exploitation. Update KACE SMA to v13.x+ and apply Microsoft's May 2026 cumulative updates for Exchange Server within 24 hours.
