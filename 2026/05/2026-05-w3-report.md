## Executive Summary

Week 3 of May 2026 highlights a shift in threat actor strategy toward the core components of modern technological sovereignty. Nation-state groups like Turla have evolved their persistence mechanisms into modular P2P architectures to evade perimeter detection, while supply chain attacks have moved 'left' into the AI development ecosystem, compromising tools like TanStack and npm to harvest secrets from major AI labs. ## Strategic Recommendations

1. **Enforce FIDO2 Hardware MFA for Developer Accounts**
   Traditional MFA is being bypassed by session theft tools used by Velvet Chollima and TeamPCP. Deploy hardware keys (e.g., YubiKey 5) to secure CI/CD pipelines and cloud consoles, effectively mitigating EvilTokens and session hijacking risks.

2. **Harden AI/ML Development Supply Chains**
   Given the attacks on OpenAI/Mistral via TanStack and npm, implement SLSA-aligned artifact verification. Use private registries and mandatory dependency pinning to prevent 'left-shift' compromises at the source level.

3. **Implement Microsegmentation for P2P Evasion**
   The Kazuar P2P botnet uses local IPC and WebSockets to hide within internal traffic. Segment networks to prevent inter-workstation P2P flows (SMB, Named Pipes, RPC) unless strictly required for operations.

## Operational Recommendations

1. **Patch Exchange Server for CVE-2026-42897 Immediately**
   Active exploitation is ongoing. Use 'Exchange Management Shell' to verify patch status (May 2026 Update) or apply EEMS (Exchange Emergency Mitigation) to block XSS vectors on OWA interfaces.

2. **Audit VS Code and npm Lifecycle Scripts**
   Run 'npm install --ignore-scripts' for all untrusted packages and set 'task.allowAutomaticTasks: off' in VS Code. Monitor for 'node.exe' or 'code.exe' spawning child shells (cmd/powershell).

3. **Block NATS Port 14222 to Untrusted IPs**
   The KeyHunter malware uses port 14222 for exfiltration via the NATS protocol. Configure EDR/Firewall alerts for any outbound traffic on this port originating from non-authorized cloud infrastructure nodes.
