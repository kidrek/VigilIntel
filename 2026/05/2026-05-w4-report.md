## Trends & Weak Signals

**Emerging techniques:**
- AI-driven autonomous post-exploitation (LLM agents mapping internal networks, querying cloud secrets, and automating database exfiltration).
- Compromising development environments via IDE extensions (VS Code Nx Console) and embedding fake Sigstore verification provenance badges.

**Declining techniques:**
- Traditional noisy ransomware encryption on production servers, being rapidly replaced by stealthy data extortion-only operations.
- Simple password-based attacks without MFA bypass, as threat groups migrate heavily toward AiTM proxies and session hijacking.

**Geopolitical context:** Geopolitical friction in Eastern Europe and the Middle East continues to drive targeted cyber espionage. The dependence of European entities on foreign cloud and satellite intelligence providers (such as Planet Labs) highlights severe digital sovereignty issues under regulations like the US Cloud Act.

## Strategic Recommendations

1. **Enforce FIDO2 Hardware MFA and Session Binding**
   Traditional SMS and TOTP-based authentication is systematically bypassed by sophisticated Adversary-in-the-Middle (AiTM) kits such as Tycoon 2FA and Kali365. Implement FIDO2/WebAuthn hardware tokens (such as YubiKey 5 Series) or Windows Hello for Business across all user profiles. Bind OAuth access tokens cryptographically to compliant machines to render stolen session cookies useless.

2. **Implement Continuous Software Composition Analysis Gates**
   The rapid escalation of software supply chain campaigns by TeamPCP targeting developer ecosystems (e.g., Nx Console extension v18.95.0) demands automated defenses. Deploy static application security testing (SAST) and software composition analysis (SCA) scanners within GitLab/GitHub actions. Block compilation of dependencies that are unsigned or fail to match strict SHA-256 integrity hash verification.

## Operational Recommendations

1. **Audit and Mitigate Drupal and Ghost Injections**
   Attackers are actively exploiting Drupal Core (CVE-2026-9082) and Ghost CMS (CVE-2026-26980) to execute code and distribute ClickFix malware. Immediately update Drupal instances to version 11.3.10+ and Ghost CMS to 6.19.1+. Configure Web Application Firewall (WAF) rules to detect and block SQL injection strings containing 'UNION SELECT' directed at public CMS endpoints.

2. **Block Remote Device Authorization Flows Globally**
   Phishing kits such as Kali365 actively abuse the OAuth 2.0 Device Code flow to gain persistent access to Microsoft 365 environments without prompting MFA. Execute the Microsoft Graph command to disable device code authentication globally in Entra ID for all non-essential accounts. Audit sign-in logs using KQL queries looking for AuthenticationProtocol=='deviceCode' to find compromised sessions.

## Recommandations Stratégiques

1. **Forcer le MFA FIDO2 et le liage de session**
   Les mécanismes d'authentification classiques par SMS ou TOTP sont contournés de manière automatisée par des kits de phishing de l'homme du milieu (AiTM) comme Tycoon 2FA ou Kali365. Déployez impérativement des clés physiques FIDO2/WebAuthn (ex: YubiKey 5) ou Windows Hello pour l'ensemble des postes de travail. Liez de manière cryptographique les jetons OAuth d'accès Entra ID aux machines de confiance pour neutraliser le vol de cookies de session.

2. **Intégrer des contrôles SCA continus dans la CI/CD**
   L'intensification des attaques de chaîne logistique orchestrées par TeamPCP ciblant les développeurs (via l'extension Nx Console v18.95.0 ou Laravel-Lang) nécessite des barrières automatisées. Intégrez des solutions de Software Composition Analysis (SCA) dans les pipelines de compilation GitHub Actions. Bloquez l'importation de paquets npm/PyPI non signés cryptographiquement ou dont les hashes SHA-256 ne correspondent pas au fichier lock d'origine.

## Recommandations Opérationnelles

1. **Corriger et mitiger les failles CMS d'urgence**
   Des attaquants exploitent activement Drupal Core (CVE-2026-9082) et Ghost CMS (CVE-2026-26980) pour exécuter du code et distribuer des malwares de type ClickFix. Mettez à jour immédiatement vos CMS vers les versions corrigées (Drupal 11.3.10+ et Ghost 6.19.1+). Configurez des règles de pare-feu applicatif (WAF) pour inspecter et bloquer les requêtes web contenant des motifs d'injections SQL 'UNION SELECT' ciblées.

2. **Désactiver globalement le protocole Device Code OAuth**
   Des kits d'hameçonnage industriels comme Kali365 exploitent abusivement le flux d'autorisation par code d'appareil pour s'emparer de comptes Microsoft 365. Exécutez les commandes d'administration Microsoft Graph pour désactiver le Device Code flow au sein de l'annuaire Entra ID pour l'ensemble des profils. Utilisez des requêtes KQL d'audit de connexion ciblant le protocole 'deviceCode' pour repérer d'éventuelles sessions compromises.
