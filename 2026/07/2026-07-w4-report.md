## Strategic Recommendations

1. **Enforce FIDO2 Hardware MFA for All Administrative Accounts**
   Traditional SMS/TOTP MFA is bypassed by Device Code phishing campaigns deployed by actors like Kali365. Enforce WebAuthn/FIDO2 hardware keys (such as YubiKey 5) across Entra ID and cloud management portals. Immediately restrict Device Code authorization flows in Entra ID via Conditional Access policies.

2. **Establish Strict Vendor Risk Governance for SaaS Supply Chains**
   Compromises at vendors like Craneware, Unlimited Systems, and groupware suppliers demonstrate severe supply chain exposure. Mandate SOC 2 Type II audits, continuous external attack surface monitoring, and mandatory breach notification SLA thresholds under 24 hours for all third-party software and SaaS integrations.

## Operational Recommendations

1. **Patch Critical SharePoint CVE-2026-50522 and Rotate Machine Keys**
   Apply the Microsoft July 2026 security updates for SharePoint Server on-premises immediately. Beyond patching, perform a full manual rotation of IIS and SharePoint machine keys on exposed nodes, as patching alone does not invalidate machine keys stolen prior to update deployment. Audit ASP.NET logs for deserialization anomalies.

2. **Isolate AI Analytics Servers and Audit Unauthenticated APIs**
   Update Langflow to version 1.3.0+ (CVE-2025-3248) and restrict network access to marimo analytics endpoints (CVE-2026-39987). Implement network isolation and strict authentication on all AI pipeline endpoints to prevent automated EncForge ransomware attacks and agentic exploitation.

## Tendances & Signaux Faibles

**Techniques émergentes :**
- Hameçonnage OAuth Device Code
- Exécution autonome d'exploits par agents IA
- Rançongiciels ciblant les artefacts d'IA (modèles et bases vectorielles)
- Détournement de SDK proxys résidentiels sur Smart TV

**Techniques en déclin :**
- Protection MFA classique par SMS/TOTP seul
- Pièces jointes Office avec macros traditionnelles

**Contexte géopolitique :** Les tensions en Asie de l'Est stimulent l'espionnage nord-coréen contre les éditeurs sud-coréens. En parallèle, les groupes RaaS d'Europe de l'Est continuent de cibler les infrastructures critiques et la santé occidentales.

## Recommandations Stratégiques

1. **Imposer le MFA matériel FIDO2 pour les comptes administratifs**
   Les mécanismes MFA traditionnels (SMS, TOTP) sont contournés par Device Code Phishing (campagnes Kali365). Déployez des clés matérielles FIDO2/WebAuthn (type YubiKey 5) sur Entra ID et les portails cloud, et restreignez le flux Device Code via des politiques d'accès conditionnel.

2. **Établir une gestion stricte des risques tiers sur la chaîne d'approvisionnement**
   Les compromissions de fournisseurs (Craneware, éditeurs de groupware) soulignent l'exposition de la supply chain. Exigez des audits SOC 2 Type II, une cartographie continue des surfaces d'attaque et des SLA de notification de violation sous 24 heures.

## Recommandations Opérationnelles

1. **Patcher SharePoint CVE-2026-50522 et renouveler les clés machine IIS**
   Appliquez immédiatement les correctifs Microsoft de juillet 2026 pour SharePoint Server. Effectuez ensuite une rotation complète des clés machine IIS/SharePoint sur les nœuds exposés, le patch seul n'involuant pas les clés volées avant mise à jour. Auditez les logs ASP.NET.

2. **Isoler les serveurs d'analytique IA et auditer les API non authentifiées**
   Mettez à jour Langflow en version 1.3.0+ (CVE-2025-3248) et restreignez l'accès aux endpoints marimo (CVE-2026-39987). Implémentez une isolation réseau et une authentification stricte sur les pipelines IA pour contrer le rançongiciel EncForge.
